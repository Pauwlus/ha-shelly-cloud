
# Shelly Cloud Home Assistant Integration (Modern)

![hacs_badge](https://img.shields.io/badge/HACS-Default-orange)

Modern integration for Shelly Cloud.

## Features
- Shelly Cloud login
- Async device polling
- Power sensors
- Config Flow UI
- Home Assistant 2024/2025 compatible
- HACS install ready

## Installation
1. Add this repository to HACS → Integrations → Custom Repositories  
   Type: Integration
2. Install the integration
3. Restart Home Assistant
4. Add integration via Settings → Devices & Services → Add Integration → Shelly Cloud

## Configuration
Provide your Shelly Cloud email and password in the config flow.
